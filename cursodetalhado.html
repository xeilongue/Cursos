Sobre o Curso

Este curso de programação é ideal para iniciantes que desejam aprender os conceitos fundamentais de programação e se preparar para o desenvolvimento de software em várias linguagens.

Conteúdo Programático

  • Introdução à programação e suas aplicações
  • Lógica de programação e resolução de problemas
  • Fundamentos de algoritmos e estruturas de controle
  • Desenvolvimento de pequenos programas em Python e JavaScript

Requisitos

Não há pré-requisitos para este curso. Todos os conceitos são ensinados do zero.

Duração

O curso tem uma duração de 40 horas, divididas em aulas teóricas e práticas.
